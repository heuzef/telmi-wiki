# Musiques libres de droits et effets sonores

Si vous cherchez des musiques sympas pour mettre un fond sonore sur vos créations, optez pour les musiques libres de droit ou sous licence Creative Commons. 

Voici une sélection de sites où vous pourrez trouver ce genre de morceaux. La plupart sont gratuits. Faites donc bien attention aux licences.

N'hésitez pas à contribuer pour actualiser cette liste pour aider les autres créateurs.
